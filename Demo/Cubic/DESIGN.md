# Cubic - 方块世界的冒险之旅

> 纯方块风格的RPG游戏，包含多种玩法模式

---

## 一、项目概述

### 1.1 项目定位

**Cubic** 是一款纯方块（Cube/Voxel）美术风格的RPG游戏，采用Unity引擎开发。游戏核心特色是：
- **纯方块视觉**：所有角色、场景、道具均使用立方体构建
- **多玩法融合**：结合RPG、冒险、解谜、建造等多种玩法
- **模块化设计**：每个玩法模块独立，可扩展性强

### 1.2 核心玩法

| 玩法类型 | 描述 | 优先级 |
|---------|------|--------|
| RPG冒险 | 主线剧情、角色成长、技能系统 | P0 |
| 探索收集 | 探索地图、收集物品、发现秘密 | P0 |
| 战斗系统 | 即时战斗、回合制战斗、BOSS战 | P0 |
| 建造系统 | 自由建造、合成系统 | P1 |
| 解谜系统 | 机关解谜、逻辑谜题 | P1 |
| 多人联机 | 合作冒险、竞技对战 | P2 |

### 1.3 目标用户

- 喜欢RPG游戏的玩家
- 喜欢方块/Minecraft风格的玩家
- 喜欢探索和收集的玩家

---

## 二、游戏世界观

### 2.1 世界观设定

**立方世界（Cubic World）**

- 由无数方块构成的神奇世界
- 不同材质方块组成各种地形：草地、泥土、石头、水、岩浆
- 生物由方块构成，拥有独特的"方块人"文明

### 2.2 主要阵营/势力

| 阵营 | 描述 | 特点 |
|-----|------|------|
| 方块王国 | 玩家起始阵营 | 正义、秩序 |
| 荒野部落 | 游牧民族 | 自由、生存 |
| 机械都市 | 科技势力 | 机械、科技 |
| 神秘学派 | 魔法势力 | 魔法、探索 |
| 暗影军团 | 反派势力 | 黑暗、征服 |

## 三、RPG系统设计

### 3.1 角色系统

**角色创建**
- 外观自定义（单一纯色方块）
- 职业选择（颜色决定职业）
- 属性分配

**职业系统 - 方块颜色标识**

> 详细职业设计请查看：[CLASSES.md](CLASSES.md)

| 职业 | 颜色 | 代码 | 定位 |
|-----|------|------|------|
| 战士（Warrior） | 铁灰色 | `#708090` | 近战物理 |
| 魔法师（Mage） | 紫色 | `#9932CC` | 远程魔法 |
| 弓箭手（Archer） | 绿色 | `#32CD32` | 远程物理 |
| 圣骑士（Paladin） | 蓝色 | `#4169E1` | 近战辅助 |
| 刺客（Assassin） | 黑色 | `#1C1C1C` | 近战爆发 |
| 工程师（Engineer） | 橙色 | `#FF8C00` | 远程召唤 |
| 死灵法师（Necromancer） | 深紫色 | `#4B0082` | 召唤/诅咒 |
| 圣职者（Cleric） | 金色 | `#FFD700` | 治疗/辅助 |

**属性系统**
- 力量（STR）：物理攻击、负重
- 敏捷（AGI）：速度、闪避
- 智力（INT）：魔法攻击、技能效果
- 体力（VIT）：生命值、防御
- 运气（LUC）：暴击、掉率

### 3.2 技能系统

**技能分类**
- 主动技能：消耗魔法值，战斗中释放
- 被动技能：永久生效，提升属性或能力
- 特殊技能：特定条件触发

**技能树设计**
- 每个职业有独立的技能树（详见 CLASSES.md）
- 技能点通过升级获得
- 可学习多个技能但有数量限制

**技能特效集成**
- 使用 EffectsManager 播放 VFX 特效
- 屏幕闪光反馈（受伤、暴击、治疗等）
- 具体特效清单见 CLASSES.md

### 3.3 成长系统

**等级系统**
- 初始等级：1级
- 最高等级：100级（可扩展）
- 经验值获取：战斗、任务、探索

**装备系统**
- 武器、防具、饰品
- 品质：普通、优秀、稀有、史诗、传说
- 套装效果

## 四、关卡/玩法模块设计

### 4.1 主线剧情关卡

**章节设计**
- 第1章：方块王国的危机
- 第2章：荒野之旅
- 第3章：机械都市的秘密
- 第4章：神秘学派的试炼
- 第5章：对抗暗影军团

**关卡类型**
- 剧情关卡：推进故事
- 战斗关卡：遭遇战
- 解谜关卡：机关谜题
- BOSS关卡：大型敌人

### 4.2 探索系统

**地图设计**
- 无缝大世界
- 多层地下城
- 隐藏区域

**收集要素**
- 收藏品（方块标本）
- 装备图纸
- 配方书
- 情报/日记

### 4.3 建造系统

**建造功能**
- 自由放置方块
- 建造防御工事
- 制作陷阱机关
- 装饰基地

**合成系统**
- 材料收集
- 配方研究
- 制作装备/道具

### 4.4 解谜系统

**谜题类型**
- 机关谜题：按钮、开关、移动方块
- 逻辑谜题：密码、图案、顺序
- 时间谜题：在限制时间内完成
- 组合谜题：结合多种元素

---

## 五、战斗系统设计

### 5.1 战斗模式

**即时战斗模式（Action RPG）**
- 玩家控制移动和攻击
- 技能快捷键释放
- 闪避、格挡机制

**回合制战斗模式（Turn-based）**
- 经典回合制
- 策略性强
- 技能组合

### 5.2 敌人设计

**普通敌人**
- 哥布林方块人
- 石像鬼方块
- 骷髅方块兵

**精英敌人**
- 各章节精英怪
- 特殊技能和弱点

**BOSS敌人**
- 各章节最终BOSS
- 多阶段战斗
- 独特机制

## 六、技术实现

### 6.1 美术资源

**方块素材**
- 基础方块：16x16x16像素风格
- 角色方块：类似RPG角色造型
- 道具方块：可识别图标设计

**动画系统**
- 行走动画：4-8帧
- 攻击动画：技能动作
- 待机动画：呼吸效果

### 6.2 框架利用

基于 WulaGameFramework 框架开发：
- CharacterManager：角色系统
- EntityManager：实体和AI
- UIManager：界面系统
- InventoryManager：背包系统
- QuestManager：任务系统（待扩展）
